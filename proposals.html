Export Salesforce report for bulk import

Use the Enterprise Proposals – Quick Import report, then upload the CSV below.

  1. Open the report. Use the link under the import area or open it in a new tab. Sign in to Salesforce if prompted.
  2. Edit the report. In Lightning, switch to edit/customize mode so you can change filters (often Edit from the report actions menu—wording may vary slightly by org).
  3. Set your Solution Number. Under Filters, edit GPC Solution: GPC Solution Number, enter your quote’s solution ID (for example Q-40776), then Apply.
  4. Run the report. Click the blue Run button so the detail rows refresh with your solution.
  5. Export as Details-only CSV. Open Export, choose Details Only, set Format to Comma Delimited (.csv), then click Export.
  6. Import here. Drag the downloaded file onto the dashed box above (or browse). Your locations and lines populate automatically.
